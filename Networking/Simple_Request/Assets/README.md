## To make and run this challenge:

**1) Have a linux based terminal open (WSL for windows works)** <br>
**2) Run the `make` command. this will compile the source code** <br>
**3) Find an ununsed port such as 8080 and note it down** <br>
**4) Run the executable: `./server 8080`** <br>
**5) Open a new linux based terminal** <br>
**6) Send your requests to the server from here using: `printf '<YOUR_HTTP_REQUEST>' | nc 127.0.0.1 8080`**
